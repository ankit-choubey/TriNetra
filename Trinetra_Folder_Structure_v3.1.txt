Important GitHub Workflow Rules
1. DO NOT push node_modules, target/, or venv folders. Add them to .gitignore immediately.
2. The 'agents' folder has ONE requirements.txt and ONE Dockerfile, but multiple entrypoints (main.py). In docker-compose, you will define 11 agent services that all build from the same image but run different commands: command: python agents/doc-agent/main.py
3. Create 4 main branches: dev-frontend, dev-backend, dev-ai-ankit, dev-ai-utkarsh.
HINGLISH: Yeh structure exactly apne repo mein copy-paste karo. Har agent ka apna folder hai, par dependencies (requirements.txt) shared hain taaki Docker image jaldi build ho. shared/ucso_schema.json ko root mein rakhna bohot zaroori hai.
